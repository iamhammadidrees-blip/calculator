Calculator Script Workflow

Step 1: Prompt for the first number using prompt("Enter the first number:"); and convert it to a float with parseFloat(), storing it in num1.

Step 2: Prompt for the second number using prompt("Enter the second number:"); and convert it to a float, storing it in num2.

Step 3: Prompt for the operator using prompt("Choose an operation:\n+ for addition\n- for subtraction\n* for multiplication\n/ for division"); and store the input in operator.

Step 4: Use a switch(operator) statement to perform the chosen calculation (+, -, *, or /) and assign the result to result.

Step 5: Output the final expression and its value by calling console.log(`${num1} ${operator} ${num2} = ${result}`);.

This script follows a procedural style and doesn’t declare any classes, constructors, or use . The main design concepts in play are: • Encapsulation: each function has a single responsibility (input validation, operator selection, calculation, or output).
